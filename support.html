PVS Support

There are many forms of support for PVS: email, mailing lists, and an associated Google group.

PVS Issues

Use one of the following to report issues, make feature requests, or suggestions for improvement.

  • pvs-sri@csl.sri.com

    This is a way to contact the PVS developers at SRI directly. If you have questions about licensing, concerns about security, requests for new features, suggestions for improving web pages, and so forth, use this address.

  • pvs-bugs@csl.sri.com

    This is used for reporting bugs or problems you experience with PVS, both in the system and the documentation. When reporting a bug, please include any specs needed to recreate it. If running the Emacs version, you can use the M-x dump-pvs-files command to create a file containing the specification and proof files suitable for mailing. But you can also send a tar, zip, or any other form to send the bug report; whatever is most convenient.

    See the reporting bugs web page for more information on how to report a bug.

  • Github issues

    PVS is hosted on Github, so the issues facility can also be used to report bugs. Note that issues are restricted in the kinds of attachments that can be accepted; it includes most graphics formats (PDF, PNG, etc.) as well as ZIP and GZ.

Google Groups

We have set up a PVS Google Groups, that can be used as a forum for discussions involving PVS, to ask for help, etc.

Mailing Lists

We maintain two mailing lists relating to PVS, pvs@csl.sri.com, and pvs-help@csl.sri.com. These are moderated to prevent spam, and are generally low volume. The lists use Mailman, the GNU Mailing List Manager.

It is recommended that users subscribe to the PVS mailing list, in order to find out about important announcements, including new versions of PVS, updated documentation, and related conferences.

pvs@csl.sri.com - archive: https://mls.csl.sri.com/cgi-bin/mailman/private/pvs/ This list is for the discussion of topics of general interest to all PVS users. This list is typically low-traffic, and contains items such as calls for papers, announcements of new PVS related papers, and notifications of new releases of PVS.

pvs-help@csl.sri.com - archive: https://mls.csl.sri.com/cgi-bin/mailman/private/pvs-help/ A moderated list for submitting questions or problems about any aspect of the use of PVS. Messages are distributed to anyone who is subscribed to this list. There are a group of experienced users already subscribed, including all the PVS developers. Note that you do not need to subscribe to ask a question, and you are under no obligation to answer questions if you do subscribe.

Subscribing and Unsubscribing To subscribe or unsubscribe to these lists, either visit http://lists.csl.sri.com/mailman/listinfo/, or send mail to the desired list with -subscribe or -unsubscribe appended (e.g., pvs-subscribe or pvs-help-subscribe, ). You must include a valid email address. After submitting your request, you will get mail asking for confirmation; just follow the instructions.
